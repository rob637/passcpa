# HeyGen Video Creation Workflow

## You have 5 videos to create. Do each one separately:

### Folder Contents:
- `backgrounds/` - 5 PNG images (one per video)
- `scripts/` - 5 TXT files (one per video)

### For EACH video:

1. **Go to HeyGen â†’ Create Video** (regular mode, NOT bulk)

2. **Add Avatar** - Click to add your avatar (Sarah/Freja)

3. **Set Background**:
   - Click on the canvas background
   - Choose "Upload" 
   - Select the matching PNG from `backgrounds/`

4. **Paste Script**:
   - Open the matching `.txt` file from `scripts/`
   - Copy all text
   - Paste into HeyGen's script box

5. **Generate** - Click Generate Video
