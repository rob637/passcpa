## QUICK START: VYOND STUDIO BASICS

### Step 1: Create New Project
1. Log into Vyond Studio (https://www.vyond.com)
2. Click **Create New Video**
3. Choose **Contemporary** style (most professional for business/tax content)
4. Set **Widescreen 16:9** and **1080p**

### Step 2: Choose Your Character
1. Go to **Characters** in left panel
2. Search: "professional" or "business"
3. Recommended: Female or male in business casual
4. Click to add to scene, then right-click ‚Üí **Save to Favorites** (reuse across scenes)

### Step 3: Scene Building Workflow
For each scene below:
1. Click **+ Add Scene** 
2. Set scene duration (noted in seconds)
3. Add background from **Props ‚Üí Backgrounds**
4. Add character, set action
5. Add text boxes (copy from "On-Screen Text" below)
6. Add audio (record or upload)

### Step 4: Audio Options
**Option A - Record in Vyond:**
- Click **Audio** ‚Üí **Record** 
- Read the script for each scene

**Option B - Upload Pre-recorded:**
- Record all audio externally (recommended for quality)
- Click **Audio** ‚Üí **Upload**
- Use script file: `audio_script.txt`

**Option C - Vyond Text-to-Speech:**
- Click **Audio** ‚Üí **Text to Speech**
- Paste script, select voice

## VYOND PRO TIPS

### Timing Text Animations
- Headline: Appear at 0.5s
- Bullet 1: Appear at 2s
- Bullet 2: Appear at 4s
- (Continue 2s apart)
- Match to when narrator says each item

### Character Actions (Vyond Contemporary)
- "Explaining" - hands gesturing
- "Presenting" - pointing at content
- "Thinking" - for pause moments
- "Celebrating" - for closing/success

### Transitions
- Use "Fade" between most scenes (0.5s)
- Use "None" within same topic
- Use "Wipe" for major section changes

### Text Styling
- Headers: 48-64pt, Bold, Dark blue
- Body: 32-40pt, Regular, Dark gray
- Callouts: 28-36pt, in colored box
- Formulas: Monospace font if available

### Export Settings
- Format: MP4
- Resolution: 1080p
- Frame rate: 30fps
