```

HOOK (first 3 seconds): "Price vs Usage Variance" then "It's PURE logic"

KEY CONTENT:
- PURE method:
  - Price variance = (Actual P - Standard P) × Actual Quantity
  - Usage variance = (Actual Q - Standard Q) × Standard Price
- Why this works:
  - Price uses ACTUAL quantity (already bought it)
  - Usage uses STANDARD price (isolate efficiency)
```
