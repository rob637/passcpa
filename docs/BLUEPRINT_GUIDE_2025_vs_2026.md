## üìã Quick Reference: What Changed

### Major Changes (REG & TCP Only)

| Topic | 2025 Blueprint | 2026 Blueprint |
|-------|---------------|----------------|
| **SALT Cap** | $10,000 | **$40,000** |
| **QBI Deduction (Sec 199A)** | 20% | **23%** |
| **Bonus Depreciation** | 40% | **100%** |
| **Social Security Tax** | Up to 85% taxable | **New exemptions** |
| **Standard Deduction** | TCJA amounts | **Permanent enhanced** |
| **Child Tax Credit** | $2,000/child | **Enhanced** |
| **AMT Exemption** | 2025 amounts | **Permanent** |
| **Estate Exemption** | ~$13.6M (expiring) | **Permanent high** |
| **Business Interest (163(j))** | ATI (EBIT) basis | **EBITDA basis** |

### Minimal Changes (FAR, AUD, BAR, ISC)

These sections have **minimal substantive changes** between 2025 and 2026 Blueprints:

- **FAR**: Same content - CECL, ASC 606, leases, etc.
- **AUD**: Same content - SQMS fully effective in both
- **BAR**: Same content - Data analytics, IFRS differences
- **ISC**: Same content - Minor AI/automation emphasis increase

## ‚ùì FAQ

### Q: I'm testing in June 2026. Should I wait until July to take REG/TCP?

**Consider:**
- ‚úÖ Study existing 2025 content now, test before July
- ‚ö†Ô∏è If you fail and retake after July, you'll need to learn new rules
- Decision factors: Your readiness, pass likelihood, study time available

### Q: Do FAR, AUD, BAR, or ISC have different blueprints?

**Minimal differences.** These sections have the same substantive content in both 2025 and 2026 Blueprints. Focus on core concepts.

### Q: I'm using a review course. How do I know which Blueprint it covers?

- Check for "2026 Blueprint" or "OBBBA/H.R. 1" mentions
- If the course mentions $40K SALT cap, 23% QBI ‚Üí It's 2026 content
- If the course mentions $10K SALT cap, 20% QBI ‚Üí It's 2025 content
- **Ask your provider** which Blueprint edition you have

### Q: What if I answer based on the wrong Blueprint?

- The exam tests based on **your testing date**
- Using wrong numbers = wrong answers
- This is especially critical for calculations
