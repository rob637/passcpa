## Content Quality Standards

### Question Quality Criteria

Every question must:

1. **Be Blueprint-Aligned**: Map to specific CFP Board topic
2. **Include Rich Explanation**: 
   - Why correct answer is correct
   - Why each distractor is incorrect
   - Real-world application tip
3. **Have Authoritative Reference**: IRC section, regulation, CFP Board standard
4. **Use Realistic Scenarios**: Actual client situations, not abstract
5. **Test Higher-Order Thinking**: Application/Analysis, not just recall
6. **Be Calculator-Appropriate**: TVM questions use realistic numbers

### Lesson Quality Criteria

Every lesson must include:

1. **Learning Objectives**: 3-5 measurable outcomes
2. **Core Content**: Clear, concise explanation with examples
3. **Practical Application**: Real-world scenarios
4. **Formulas**: Clearly formatted with examples
5. **Mnemonics**: Memory aids for key concepts
6. **Cross-References**: Links to related topics
7. **Practice Problems**: 3-5 embedded questions
8. **Key Takeaways**: Bulleted summary

### Case Study Quality Criteria

1. **Realistic Demographics**: Believable family situations
2. **Complete Financial Picture**: No missing data for solvable questions
3. **Integrated Problems**: Questions require connecting multiple concepts
4. **Progressive Difficulty**: Mix of straightforward and nuanced questions
5. **Updated Figures**: Current tax years, contribution limits, etc.
