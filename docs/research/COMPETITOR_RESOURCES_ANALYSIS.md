# Competitor CPA/CFP Exam Prep Resources Analysis

**Date:** February 2026  
**Purpose:** Identify best practices from leading exam prep providers to improve our Resources page

---

## Executive Summary

Analysis of 6 major competitors (Becker CPA, UWorld Roger, Surgent CPA, Wiley CPAexcel, Dalton CFP, Kaplan CFP) reveals common patterns in how they structure study resources. Key findings include categorized formula sheets, "Why Take This Exam" value propositions, adaptive study materials, and comprehensive quick reference guides.

## 2. Common Patterns Identified

### 2.1 Content Types (Most to Least Common)

| Resource Type | Frequency | Description |
|---------------|-----------|-------------|
| Practice Questions | 6/6 | Everyone has this |
| Video Lectures | 6/6 | Core learning content |
| Flashcards | 5/6 | Digital, sometimes printable |
| Quick Reference Sheets | 5/6 | Condensed "cheatsheets" |
| Practice Exams | 5/6 | Full exam simulations |
| Formula Sheets | 4/6 | Usually categorized by topic |
| Audio Content | 3/6 | Mobile learning support |
| Study Calendars | 4/6 | Personalized study schedules |
| Mnemonics | 3/6 | Integrated into lectures or separate |

### 2.2 Organization Patterns

1. **Section-Based Primary, Topic-Based Secondary**
   - CPA: FAR, AUD, REG, Discipline ‚Üí Then by topic within section
   - CFP: Insurance, Tax, Investments, Retirement, Estates, Fundamentals

2. **Tile/Card-Based Navigation**
   - Large clickable tiles for main resource categories
   - Visual icons and descriptions
   - Item counts or progress indicators

3. **Progress Integration**
   - Resources tied to overall study progress
   - Completion badges or checkmarks
   - "Recommended for you" suggestions

### 2.3 "Exam Value" Sections

**What competitors include:**
- Career benefits and salary data
- Pass rate statistics
- Student testimonials
- "Why pursue this certification" messaging
- Comparison with non-certified peers

**Where it appears:**
- Dedicated landing pages
- Within course introduction
- Marketing materials and homepage

## 3. Recommendations for Our Resources Page

### 3.1 Quick Wins (Implement Immediately)

1. **Add "Why Take This Exam" Section**
   ```
   - For CPA: Salary advantages, career opportunities, state requirements
   - For EA: IRS representation, year-round practice, lower barrier to entry
   - For CFP: Client trust, income increase, professional credibility
   - Include statistics (e.g., "CPAs earn 10-15% more than non-CPAs")
   ```

2. **Categorize Formula Sheets by Topic**
   - Instead of one massive formula sheet, break into:
     - CPA FAR: Govt/NFP Formulas, Financial Statement Formulas, Lease Calculations
     - CPA REG: Tax Rate Tables, Basis Calculations, Entity Comparison Formulas
   - Users can access relevant sections without scrolling through irrelevant content

3. **Add Quick Reference Tables**
   - Deadline tables (filing dates, extension dates)
   - Comparison charts (entity types, audit opinion types)
   - Threshold tables (materiality, dollar limits)

### 3.2 Medium-Term Improvements

4. **Exam Readiness Assessment**
   - Quiz that estimates preparedness
   - Suggests which resources to focus on
   - Builds user confidence

5. **Study Schedule Templates**
   - 3-month, 6-month, 9-month plans
   - Customizable based on target exam date
   - Integrate with calendar apps

6. **Audio/Podcast Versions**
   - Audio summaries of cheatsheets
   - Ideal for commute study time
   - Dalton competitors offer this feature

### 3.3 UX Improvements

7. **Visual Hierarchy Enhancement**
   - Larger tiles with icons for main categories
   - Progress indicators on each resource category
   - "Recently Viewed" quick access

8. **Mobile-First Quick Access**
   - Offline-capable cheatsheets
   - Swipe navigation between related resources
   - "Study Mode" with distraction-free viewing

9. **Search and Filter**
   - Search across all resources
   - Filter by section, topic, resource type
   - Bookmark/favorite system

### 3.4 Unique Differentiators to Consider

10. **AI-Powered Study Recommendations**
    - Based on practice performance, suggest resources
    - "You struggled with leases, here's the Lease Quick Reference"

11. **Community-Contributed Tips**
    - User-submitted mnemonics (moderated)
    - Upvoting system for best tips
    - Builds engagement and community

12. **Interactive Formula Calculator**
    - Input values, calculate results
    - Shows step-by-step with formula
    - Practice tool integrated into formula sheets

| Paywalled resources | Free basic resources, premium for full access |
