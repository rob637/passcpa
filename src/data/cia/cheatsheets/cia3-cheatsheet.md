## Domain II: Information Security (25%)

### CIA Triad
| Element | Description |
|---------|-------------|
| Confidentiality | Prevent unauthorized disclosure |
| Integrity | Prevent unauthorized modification |
| Availability | Ensure authorized access |

### Defense in Depth Layers
1. Physical (facilities, locks)
2. Network (firewalls, segmentation)
3. Host (hardening, patching)
4. Application (secure coding)
5. Data (encryption, classification)

### Access Control Principles
| Principle | Description |
|-----------|-------------|
| Least Privilege | Minimum necessary access |
| Need to Know | Access based on role |
| Separation of Duties | Split sensitive functions |
| Defense in Depth | Multiple control layers |

### Authentication Factors
| Factor | Example |
|--------|---------|
| Something you know | Password, PIN |
| Something you have | Token, smart card |
| Something you are | Biometrics |
| Somewhere you are | Location |

### Common Attack Vectors
| Attack | Target |
|--------|--------|
| Phishing | Human (email deception) |
| Malware | Systems (malicious code) |
| Ransomware | Data (encryption for ransom) |
| SQL Injection | Applications (database) |
| DDoS | Availability (flood traffic) |

### Incident Response Phases
1. **Preparation** - Plans, training, tools
2. **Identification** - Detect and analyze
3. **Containment** - Limit damage
4. **Eradication** - Remove threat
5. **Recovery** - Restore operations
6. **Lessons Learned** - Improve

### Data Classification Levels
| Level | Handling |
|-------|----------|
| Public | No restrictions |
| Internal | Standard protection |
| Confidential | Enhanced controls |
| Restricted | Strictest protection |

## Domain IV: Financial Management (20%)

### Time Value of Money
| Concept | Formula Use |
|---------|-------------|
| Present Value | Future value ÷ (1+r)^n |
| Future Value | Present value × (1+r)^n |
| NPV | Sum of discounted cash flows - investment |
| IRR | Discount rate where NPV = 0 |

### Capital Budgeting Methods
| Method | Approach |
|--------|----------|
| NPV | Discount cash flows, compare to investment |
| IRR | Find rate where NPV = 0 |
| Payback | Time to recover investment |
| ROI | Return relative to cost |

### Financial Ratios
| Category | Ratios |
|----------|--------|
| Liquidity | Current ratio, quick ratio |
| Profitability | ROA, ROE, profit margin |
| Leverage | Debt-to-equity, interest coverage |
| Efficiency | Asset turnover, inventory turnover |

### Key Ratios Formulas
| Ratio | Formula |
|-------|---------|
| Current Ratio | Current Assets ÷ Current Liabilities |
| Quick Ratio | (Current Assets - Inventory) ÷ Current Liabilities |
| Debt-to-Equity | Total Debt ÷ Total Equity |
| ROE | Net Income ÷ Shareholders' Equity |
| ROA | Net Income ÷ Total Assets |

### Budgeting Approaches
| Type | Description |
|------|-------------|
| Incremental | Adjust prior year |
| Zero-Based | Justify all expenses |
| Flexible | Adjust for activity level |
| Rolling | Continuously updated |

### Cost Concepts
| Type | Description |
|------|-------------|
| Fixed | Don't vary with activity |
| Variable | Vary with activity |
| Direct | Traceable to product |
| Indirect | Allocated (overhead) |
| Sunk | Already incurred, irrelevant |
| Opportunity | Foregone alternative |

### Working Capital
**Working Capital = Current Assets - Current Liabilities**

| Component | Management Focus |
|-----------|------------------|
| Cash | Optimize balances |
| Receivables | Speed collection |
| Inventory | Balance costs |
| Payables | Manage timing |
